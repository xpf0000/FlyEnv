# FlyEnv新版本更新日志

本次更新内容：
1. 添加了n8n模块
2. OpenClaw模块新增配置文件Tab，可以直接查看编辑配置文件。
3. OpenClaw模块新增全部的命令的快捷操作
4. 修复fnm和nvm原来无法正常使用的问题， 现在可以正常操作了。

n8n模块的PR：https://github.com/xpf0000/FlyEnv/pull/584
由ibraimfarag提交，记得感谢下他。
